> 반려동물을 입양하고 소품을 거래하는 플랫폼입니다.

## Build Setup

프로젝트를 다운받은 뒤, 사용 중이신 프레임워크에서 아래와 같은 순서로 실행해주세요.

npm install > npm run dev
